# 摸鱼工厂 — GDD（一页版）

## 概述

| 项 | 内容 |
|----|------|
| 名称 | 摸鱼工厂 |
| 平台 | 抖音小游戏（竖屏） |
| 类型 | Idle / 增量 |
| 目标会话 | 3–8 分钟/次，适合碎片化 |
| MVP 周期 | 4 周 |

## 电梯_pitch

你是工厂里最爱摸鱼的员工。点击屏幕「摸鱼」赚取 **摸鱼值**，升级自动摸鱼设备，离线也能积累收益。看一条短视频，全厂双倍摸鱼 5 分钟。

## 核心循环

```
点击摸鱼 → 获得摸鱼值 → 购买升级 → 自动产出加快
                ↓
         离线积累 → 回到游戏领取
                ↓
         看激励视频 → 5 分钟双倍产出
```

## 资源

| 资源 | 说明 |
|------|------|
| 摸鱼值 | 唯一软通货，用于购买升级 |

## 升级系统（MVP 5 项）

| ID | 名称 | 效果 | 基础价格 |
|----|------|------|----------|
| click_power | 手速训练 | 点击 +1/级 | 10 |
| auto_desk | 自动工位 | 自动 +0.5/秒/级 | 50 |
| offline_cap | 摸鱼仓库 | 离线上限 +1h/级 | 200 |
| offline_bonus | 带薪摸鱼 | 离线倍率 +10%/级 | 500 |
| global_bonus | 全厂加成 | 全产出 +5%/级 | 1000 |

**价格公式**：`floor(baseCost × 1.15^level)`

## 离线收益

- 基础离线上限：**2 小时**
- 「摸鱼仓库」每级 +1 小时（最多 8 级 → 共 10 小时）
- 离线速率 = 当前自动产出（若自动为 0，按 0.1/秒 保底）
- 离线倍率受「带薪摸鱼」和「全厂加成」影响

## 变现（IAA）

| 点位 | 形式 | 奖励 |
|------|------|------|
| 双倍摸鱼 | 激励视频 | 5 分钟全产出 ×2 |
| （v1.1）离线翻倍 | 激励视频 | 离线收益 ×2 领取 |
| （v1.1）再得广告 | multiton 连看 | 阶梯金币奖励 |

抖音小游戏收入约 90% 来自激励视频；MVP 先做 1 个广告点位。

## 目标与进度（无传统「通关」）

- 短期：解锁第 2、3 个升级
- 中期：工厂等级 10（全升级合计等级 ≥ 10）
- 长期：Prestige 转生（**不在 MVP**，v2 考虑）

## UX 原则

1. **3 秒内** 理解：大按钮「摸鱼」，顶部数字上涨
2. **30 秒内** 首次升级可购买（手速训练 Lv.1 成本 10）
3. **3 分钟内** 解锁自动工位，感受「挂机」
4. 数字格式化：1.2K / 3.4M，避免长串数字

## 技术约束

- Cocos Creator 3.8，TypeScript
- 主包 < 4MB
- 存档：`tt.setStorageSync`
- 无联网、无账号、无社交（MVP）

## 成功指标（内测）

| 指标 | 目标 |
|------|------|
| 首次会话时长 | ≥ 5 分钟 |
| 次日留存 | ≥ 20%（有广告点位后测） |
| 首次升级率 | ≥ 80% 用户在 2 分钟内购买升级 |
